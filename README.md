# JLJ&LWJ
鸢尾花分类
==
数据集说明
--
这是一个公开的数据集，一共有150条数据集，每条有4个属性，分类标签一共有三类，按0-50、51-100、100-150分别标签0、1、2。  
数据集划分
--
训练集：1-30、50-80、100-130  
验证集：30-40、80-90、130-140  
测试集：40-50、90-100、140-150  
实现方法
--
使用逻辑回归对鸢尾花进行分类，需要设计三个分类器，激活函数使用sigmoid函数，迭代训练使用梯度下降的方法。  
实现工具
--
Python 2.7.12  
numpy 1.16.1  
sklearn 0.20.2
